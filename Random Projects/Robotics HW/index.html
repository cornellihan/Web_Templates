Weather detector

It can't be done with one LDR, cause one LDR can get only one sensitivity, so with one LDR, we just can find is it day or night. But if we use another LDR and give it a sensitivity where the high signal is very high sunlight and the low signal is less sunlight, then we can also find how the day is, is it very cloudy or light. So we can also use some extra LDR to get the weather more properly.

Fire Alarm

Automatic Irrigation
